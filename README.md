# The Wire 

**The Wire** is a modern, responsive news web application built with **React** and **Vite**. It fetches top headlines from [NewsAPI](https://newsapi.org/) and displays them by category, allowing users to navigate through multiple news sections such as General, Business, Sports, Technology, Health, Science, and Entertainment.  

---

## Features

- Fetches latest news from NewsAPI dynamically
- Category-based navigation (General, Business, Sports, Technology, Health, Science, Entertainment)
- Pagination for browsing multiple pages of articles
- Responsive layout with Bootstrap
- Loading spinner while fetching news
- Deployable on **GitHub Pages**
- Hash-based routing for seamless navigation on static hosting

---

## Technology Stack

- **Frontend:** React (Class Components)
- **Bundler:** Vite
- **Routing:** React Router DOM (`HashRouter` for GitHub Pages)
- **Styling:** Bootstrap
- **Deployment:** GitHub Pages
- **API:** [NewsAPI.org](https://newsapi.org/)
